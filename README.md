# Payoy Next

Payoy adalah mini company profile untuk startup digital yang membantu bisnis kuliner mengelola menu, layanan, dan kontak usaha.

## Isi Project

- Landing page: `/`
- About page: `/about`
- Services page: `/service`
- Contact page: `/contact`
- Halaman menu: `/menu`
- Halaman detail dinamis: `/menu/[slug]`

## Fitur

- Menggunakan Next.js App Router dengan folder `app/`.
- Menggunakan layout global untuk header dan footer.
- Memiliki Server Component untuk halaman detail menu.
- Memiliki Client Component untuk interaksi tambah item dan checkout.
- Mengambil data makanan dari API publik TheMealDB.
- Menggunakan komponen dasar shadcn/ui.

## Menjalankan Project

```bash
pnpm install
pnpm run dev
```
